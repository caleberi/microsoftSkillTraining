# Database management system 

Database management system  is a software used to access structured database/ collection of data.

### Why use database?

* They make data assessible,reliable and also structured.

### Components of database

* Row/Record
* Column/Field
* Table -> collections of rows and column.

NOTE: ``` A database contain more than one table ```.

However spreadsheet on give structure and help identify gaps in data.

### Benefit of Database

* Provide structure for data.
* Allow enforcement  for rules for data
* Protect data from unauthorized access or changes.

## Relational Database

This deals with  organization of table into related entity.The relationship in this database can be

* one to one 
* one to many
* many to one
* many to many


NOTE: ```ACID Transaction requires that all transaction must be completed before it can be successful .It means it must be atomic,consistent,isolated and durable```

### Rules to take into consideration when building a database

* Use an Entity Relationship Diagram
* What should to database store.
* Record all attribute related to the data
* Note the required data type .
* Use precise datatype to accomodate data and prevent underflow and  over flow
* Provide keys for identification based on uniqueness of data
